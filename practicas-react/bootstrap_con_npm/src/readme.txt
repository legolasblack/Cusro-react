empezamos esta practica con la intencion de documentar el uso de bootstrap 
ya sea utilizarlo directo en la pagina index.html o mandarlo llamar en la importacion
de la pagina principal utilizando el gestor de paquetes npm para la tarea.

1.- creamos el proyecto con nox create-react-app
2.- vamos a la pagina de bootstrap y copiamos el comando de npm 
para agregar la importacion de bootstrap(npm install bootstrap)comando copiado a la fecha 29/04/2023
3.- una vez que corra el comando podemos checar el archivo package.json donde se agregan todas las dependencias de npm 
para corroborar que este listo el paquete 
4.- Solo falta importar dentro del archivo de App.js o App.jsx o index.js con la sintaxis correspondiente a React 
import 'bootstrap/dist/css/bootstrap.min.css'
import 'bootstrap/dist/js/bootstrap.min.js'(Estas dos importaciones estan hechas y direccionadas hacia la carpeta que agrego npm desde donde se debe de mandar llamar).
happy hacking!!
